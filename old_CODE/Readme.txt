


Modify the following lines before running the code on a new machine

1) set the resolution: 
    expmnt.screenRect =  [0 0 1024 768];% for the eye tracker machine

2) set the display refresh rate : 
    expmnt.refreshRate =  85; %Hz
3) set the mode for running the code: mouse or eye-tracker}
expmnt.useEyeTracker = 1; % real eyetracker (set to 0 for mouse)
